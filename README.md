# Steps to be followed to enable audio in blueStack ---
1. Login with gmail in Bluestack and open google playstore.
2. Download 'Google' app.
3. Download 'Google Text-To-Speech' app from playstore.
4. Goto Home --> System apps --> Setings --> Accessibility --> Text-To-Speech output --> Select 'Google Text-To-Speech Engine'
5. Set default language and 'Install voice data'.Then download voice sample and select.
